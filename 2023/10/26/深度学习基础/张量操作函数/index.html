张量操作函数

发表于 2023-10-26 更新于 2024-03-12 分类于 深度学习 阅读次数：
本文字数： 3.2k 阅读时长 ≈ 3 分钟

张量维度操作函数

stack和cat

  • torch.stack: 这个函数会在指定的维度上创建一个新的维度，并将输入张量堆叠到这个新维度上。因此，堆叠后的结果的维度比输入张量多一个。你需要指定 dim 参数，以确定在哪个维度上堆叠。
  • torch.cat: 这个函数会在指定的维度上将多个输入张量拼接在一起，不会创建新的维度。它会保持输入张量的维度不变，只是在指定的维度上合并。

squeeze和unsqueeze

  • torch.squeeze 用于去除张量中大小为1的维度。如果带有参数例如squeeze(-1)会检测最后一个维度是否为1，如果为1则删除这个维度。

  • torch.unsqueeze 用于在张量的指定位置插入大小为1的新维度。

transpose和permute

  • torch.transpose 主要用于交换两个维度的位置，即将张量的维度重新排列，但不改变维度的个数。
  • torch.permute 提供了更强大的功能，可以重新排列维度的顺序。可以使用 permute 来对维度进行复杂的排列。

repeat

repeat_interleave

flatten

x = x.flatten(2)

将张量 x 沿着第二维（从0开始）进行扁平化（flatten），也就是将第二维的所有元素展平为一个维度.

具体地说，如果张量 x 的形状是 (batch_size, dim1, dim2, dim3)，那么这行代码将把 dim2 维度的所有元素展平到一起，形成一个新的维度，新的形状是 (batch_size, dim1, dim2 * dim3)。

普通操作函数

torch.meshgrid

原型：

meshgrid函数输入为张量列表和索引，张量列表为一维向量或标量（被视为尺寸为（1，）），索引为“xy”，或“ij”。

如果选择“xy”，则第一维度对应于第二输入的基数，第二维度对应于第一输入的基数。

如果选择“ij”，则维度的顺序与输入的基数相同。

tensor.reshape/view

两者都是用来重塑tensor的shape的。view只适合对满足连续性条件（contiguous）的tensor进行操作，而reshape同时还可以对不满足连续性条件的tensor进行操作，具有更好的鲁棒性。
