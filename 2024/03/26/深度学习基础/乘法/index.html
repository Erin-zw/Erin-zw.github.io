乘法

发表于 2024-03-26 分类于 深度学习 阅读次数：
本文字数： 798 阅读时长 ≈ 1 分钟

在 PyTorch 中，常用的矩阵乘法函数有以下几种：

  1. torch.mm():
    • 用于计算两个二维矩阵之间的矩阵乘法。例如，对于大小分别为 (m x n) 和 (n x p) 的两个矩阵，torch.mm(matrix1, matrix2) 返回一个 (m x p) 大小的结果矩阵。
  2. torch.matmul():
    • 是一个更为通用的矩阵乘法函数，它可以处理任意维度的张量，遵照标准的矩阵乘法规则。对于二维矩阵乘法，它的行为与 torch.mm() 相同。对于更高维度的张量，遵循广播规则和矩阵乘法的广义定义。
  3. @ 运算符:
    • 自从Python 3.5版本开始，引入了一个专门用于矩阵乘法的运算符 @。在PyTorch中，你可以直接使用这个运算符来代替 torch.matmul() 进行矩阵乘法。
  4. torch.bmm():
    • 批量矩阵乘法，用于计算两个三维张量间的矩阵乘法，其中最后一个维度被认为是矩阵的内部维度，进行矩阵乘法。
  5. 点乘（逐元素乘法）:
    • 如果你想要进行的是逐元素（element-wise）乘法，而不是矩阵乘法，可以使用 torch.mul() 或者使用 * 运算符。

请注意，对于真正的矩阵乘法（并非逐元素乘法），一定要确保输入的张量满足矩阵乘法的要求，即左矩阵的列数等于右矩阵的行数。
