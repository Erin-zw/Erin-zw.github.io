旋转图像

发表于 2024-03-26 分类于 hot/矩阵 阅读次数：
本文字数： 738 阅读时长 ≈ 1 分钟

给定一个 n × n 的二维矩阵 matrix 表示一个图像。请你将图像顺时针旋转 90 度。

你必须在原地 旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要 使用另一个矩阵来旋转图像。

我的思路：由外而内，先看第一个外圈的四个角元素，会发现顺时针旋转90°就是把左上和左下交换，左下和右下交换，右下和右上交换。由此旋转完成。接着我们把四个角元素的位置都移动一位，会发现交换原则没变，所以n-1次可以旋转完成最外层。接着向内k层，会发现要n-1-2*k次。我们总共要向内n/2-1层。注意下标即可。
