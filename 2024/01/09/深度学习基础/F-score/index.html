f-score,混淆矩阵

二分类

真实情况 预测结果
正例 反例
正例 TP FN
反例 FP TN

TP：True Positive, 被判定为正样本，事实上也是正样本。 FP：False Positive,被判定为正样本，但事实上是负样本。 TN：True Negative, 被判定为负样本，事实上也是负样本。 FN：False Negative,被判定为负样本，但事实上是正样本。

Accuracy：表示预测结果的准确度，预测正确的样本数除以总样本数。 precision: 精确率，又称为查准率，表示预测结果中，预测为正样本的样本中，正确预测为正样本的概率； recall: 召回率，又称为查全率，表示在原始样本的正样本中，最后被正确预测为正样本的概率； \[ Accuary = \frac{TP+TN}{TP+FN+FP+FN}\\ Precision = \frac{TP}{TP+FP}\\ Recall = \frac{TP}{TP+FN} \]

\[ F-Score = \frac{(1+\beta^2)precision\cdot recall}{precision+recall}\\ F1-Score = 2\frac{precision\cdot recall}{precision+recall} \]

\(\beta\rightarrow\infty,F-Score\rightarrow recall\)

\(\beta\rightarrow0,F-Score\rightarrow precision\)

多分类

混淆矩阵：把二分类行列扩展即可得到多分类混淆矩阵。

假设有150个样本数据

真实情况 预测结果
类别1 类别2 类别3
类别1 43 2 0
类别2 5 45 1
类别3 2 3 49

每一行之和表示该类别的真实样本数量，每一列之和表示被预测为该类别的样本数量。

第一行第一列中的43表示有43个实际归属第一类的实例被预测为第一类，同理，第一行第二列的2表示有2个实际归属为第一类的实例被错误预测为第二类。

准确率（Accuracy）：准确率是最常用的分类性能指标。可以用来表示模型的精度，即模型识别正确的个数/样本的总个数。一般情况下，模型的acc越高，说明模型的效果越好。

\(Accuracy = \frac{43+45+49}{150}\approx91.3\%\)

精确率（Precision）：又称为查准率，表示在模型识别为正类的样本中，真正为正类的样本所占的比例。一般情况下，查准率越高，说明模型的效果越好。

\(P_1 = \frac{43}{43+5+2},P_2 = \frac{45}{2+45+3},P_3 = \frac{49}{0+1+49}\)

召回率（Recall）：又称为查全率，召回率表现出在实际正样本中，分类器能预测出多少。

\(R_1 = \frac{43}{43+2+0},R_2 = \frac{45}{5+45+1},R_3 = \frac{49}{2+3+49}\)
